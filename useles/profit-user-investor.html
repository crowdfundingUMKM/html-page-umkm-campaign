Profit User Investor

Profit yang diperoleh oleh user investor akan dielola oleh campaign atau proyek UMKM yang telah berhasil terdanai. Jika target tercapai dan sesuai dengan kesepakatan, admin Crowdfunding UMKM akan mencairkan dana investasi kepada user investor. Proses pencairan dana ini akan dilakukan secara transparan dan aman untuk memastikan bahwa investor mendapatkan keuntungan yang telah dijanjikan.

Informasi Status:

  • Waiting: Menunggu proses dari admin apakah dana bisa di cairakan untuk user investor
  • Succses: Dana sudah diterima oleh user investor
  • Rejected: Dana ditolak, adanya kemungkinan anda belum mengisi pembayaran yang seharusnya.
